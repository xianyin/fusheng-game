🐮 牛牛家族·玩法指南

欢迎来到《牛牛家族》！这不仅仅是一个简单的低买高卖游戏，更是一部跨越世代的家族商业史诗。你需要通过经商、置业、技艺和社交，让你的家族在江南大地生生不息，从一介行脚商成长为富可敌国的江南巨贾。

一、 核心目标

1. 短期目标：活下去

关注健康：移动、劳作都会消耗健康，健康归零则本世结束。记得及时在城市中“修整”回血。

积累本金：通过集市倒买倒卖赚取差价，积累第一桶金。

2. 中期目标：身份晋升

在【资产】页面查看晋升条件。

从行脚商晋升为市井掌柜、州府商首乃至江南巨贾。

特权：更高的身份可以降低运费、解锁“垄断”功能（卖出加价），甚至开启高利润的“贡品贸易”。

3. 长期目标：世代传承

每一世都有专属的人生目标（如“初入商海”、“开枝散叶”）。完成目标可获得传家宝（强力Buff）。

当主角退休或病逝后，部分资产（70%）和技艺（50%）会传承给下一代。

利用几代人的积累，打造一个万世不朽的商业帝国。

二、 核心系统详解

🛒 1. 商贸系统

低买高卖：不同城市的商品基价不同（如北方缺茶、南方缺参）。

节气影响：

清明：新茶上市，生茶价格暴跌。

寒露：天冷了，丝绸需求上涨。

大寒：岁末宴请，酒肉价格暴涨。

提示：时刻关注顶部的节气提示！

随机市场：每个城市的集市每天只随机供应部分商品。如果集市不收某种货（列表里没有），你就无法在这里出售。

🏭 2. 产业与技艺（暴利来源）

单纯倒买倒卖利润有限，产销一体才是王道。

购买产业：在【资产】页购买桑园、茶山或铁矿。它们会随时间自动产出原材料（存入背包）。

学习技艺：在【技艺】页，消耗原材料和少量加工费，制作高价值成品。

桑蚕茧 → 苏绣丝绸 （需习得缫丝）

雨前生茶 → 西湖龙井 （需习得炒茶）

长白山参 → 回春丹 （需习得炼丹，高价药！）

熟练度：制作得越多，成功率越高。

🌏 3. 世界与探索

移动成本：跨城移动需要消耗“时间”和“运费”。身份越高，运费折扣越大。

随机奇遇：

路遇劫匪：破财免灾还是拼死一搏？（看你的体魄和运气）

落魄书生：施舍他可能获得珍贵配方。

神秘行商：低价货可能是宝物，也可能是垃圾。

💕 4. 红颜知己

结识：在【知己】页拜访各地的佳人（芸娘、苏小小、董小宛）。

羁绊：通过拜访、赠礼（她们有各自喜欢的特产）提升好感等级。

强力Buff：

好感达到 Lv3 可邀请跟随。

芸娘跟随：休息回血量翻倍（保命神器）。

苏小小跟随：房产收益 +50%（后期躺赚神器）。

董小宛跟随：背包容量 +50（跑商神器）。

三、 进阶生存法则

开局选好天赋：每一世开局的“天赋选择”至关重要。

新手推荐**“商贾世家”（多钱）或“武林世家”**（血厚）。

高手尝试**“鬼谷传人”**（技艺速成）。

囤货看时机：不要把钱全部压在一种货上，万一遇到下个城市“今日不收此货”，资金链会断裂。

房产是低保：尽早买一处“城郊别院”或“闹市铺面”。即使跑商亏了，每天的房产租金也能让你有口饭吃。

注意健康红线：健康低于 30 时非常危险，遇到恶劣天气（大暑、大寒）或劫匪可能直接暴毙。预留好住店回血的钱。

四、 常见问题

Q: 为什么我有货却卖不出去？

A: 当地集市如果不经营这种商品（列表中没有），商家是不会收购的。去别的城市碰碰运气吧。

Q: 怎么提升技艺成功率？

A: 多做。失败乃成功之母，即使失败也会增加熟练度。另外，“鬼谷传人”天赋可以提高成功率。

Q: 世代传承会继承什么？

A: 继承上一世最终资金的 70% 和所有技艺熟练度的 50%。房产和好感度需要重新经营。
